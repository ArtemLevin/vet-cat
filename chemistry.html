Наглядная химия: теория и упражнения (ЕГЭ)

Цель: подготовка к ЕГЭ • Уровень: уверенный • Фокус: логика распознавания вещества или реакции • Режим: пошаговое раскрытие.

Пошаговый разбор

Идея решения

  • Определяем класс вещества по функциональной группе или признаку.
  • Подбираем реактив, который даёт однозначный наблюдаемый эффект.
  • Проверяем, что уравнение химически правдоподобно и уравнено.
  • Отсекаем похожие, но неверные реактивы (например, «серебряное зеркало» не для большинства кислот).
1) Что дано

Нужно распознать вещества и типы реакций из тем пособия: карбоновые кислоты, альдегиды, одно- и многоатомные спирты, алкены/фенол; а также типовые неорганические обменные реакции и расчёты.

2) Что требуется найти

Определить корректный реактив и предсказать признак реакции (газ, осадок, изменение цвета), затем подтвердить записью уравнения и короткой проверкой.

3) Распознавание класса/типа

Органика: ориентируемся на функциональные группы.
Неорганика: проверяем критерий ионного обмена (осадок / газ / слабый электролит).
Расчёты: применяем стехиометрию, n = V/22,4 при н.у., формулу выхода η.

4) Что означает каждое условие

Если дан NaHCO₃, ожидаем реакцию с кислотой и выделение CO₂. Если «аммиачный раствор оксида серебра» — это тест на альдегид (исключение: HCOOH). Если Cu(OH)₂ при комнатной температуре — тест на многоатомный спирт.

CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂↑
5) Ключевая химическая идея

Используем селективный признак: реактив выбираем не «похожий», а тот, который даёт максимально однозначный результат для данного класса веществ.

6) Пошаговое доведение до ответа

Пример А (альдегид): реактив Толленса, признак — «серебряное зеркало».

R–CHO + 2[Ag(NH₃)₂]OH → R–COONH₄ + 2Ag↓ + 3NH₃ + H₂O

Пример Б (этанол): мягкое окисление оксидом меди(II).

CH₃CH₂OH + CuO → CH₃CHO + Cu + H₂O (t°)

Пример В (ОВР с HNO₃(конц.), t°):

Fe + 6HNO₃ → Fe(NO₃)₃ + 3NO₂ + 3H₂O

Пример Г (ионный переход):

Молекулярное: CuSO₄ + Na₂CO₃ + H₂O → Cu(OH)₂↓ + CO₂↑ + Na₂SO₄
Полное ионное: Cu²⁺ + SO₄²⁻ + 2Na⁺ + CO₃²⁻ + H₂O → Cu(OH)₂↓ + CO₂↑ + 2Na⁺ + SO₄²⁻
Сокращённое ионное: Cu²⁺ + CO₃²⁻ + H₂O → Cu(OH)₂↓ + CO₂↑
7) Финальная проверка по условию
  • Проверены коэффициенты и сохранение атомов.
  • Для ионного уравнения сохранён заряд и сокращены только ионы-наблюдатели.
  • Для расчётов используется корректная размерность (л, моль, г, %).
  • Для выбора реактива объяснено, почему альтернативы не подходят.
8) Краткий вывод

Метод распознавания строится на связке «класс вещества → селективный реактив → наблюдаемый признак → проверка уравнения». Этот же шаблон переносится на похожие задачи ЕГЭ.

Типичные ошибки

  • Использовать реактив Толленса для всех карбоновых кислот (кроме HCOOH это неверно).
  • Считать, что между двумя солями/кислотами в растворе реакция всегда идёт, даже без осадка/газа/слабого электролита.
  • Пропускать коэффициент 6 в уравнении Fe + HNO₃(конц.) → ...
  • В расчётах выхода путать теоретическую и практическую массу.
  • Слишком рано округлять результаты и терять точность.
